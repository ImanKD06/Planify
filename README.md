# Planify
Planify

Planify es una aplicación web diseñada para gestionar los turnos y horarios de empleados en pequeñas y medianas empresas. Su objetivo es centralizar la información, facilitar la planificación y mejorar la comunicación interna, reemplazando métodos manuales como hojas de cálculo o agendas físicas.

Con Planify, los administradores pueden crear empleados, definir turnos, asignarlos y gestionar solicitudes de cambios o días libres. Los empleados, por su parte, pueden consultar su calendario personal y enviar solicitudes de forma sencilla. La aplicación incluye roles diferenciados (administrador y empleado) y un frontend multipágina fácil de usar.
